# Competency Questions

This directory contains single and multi-bridge KG selection competency questions for the used schemas.

## Directory and File Overview

* `2bridge_currkg_core_scholar.txt` — Competency questions for bridging the Curriculum KG and rich Core Scholar schemas.
* `2bridge_currkg_kwg.txt` — Competency questions for bridging the Curriculum KG and KnowWhereGraph schemas.
* `2bridge_enslaved_core_scholar.txt` — Competency questions for bridging the Enslaved and rich Core Scholar schemas.
* `2bridge_enslaved_currkg.txt` — Competency questions for bridging the Enslaved and Curriculum KG schemas.
* `2bridge_gmo_core_scholar.txt` — Competency questions for bridging the Geolink Modular Ontology and rich Core Scholar schemas.
* `2bridge_gmo_currkg.txt` — Competency questions for bridging the Geolink Modular Ontology and Curriculum KG schemas.
* `2bridge_kwg_enslaved.txt` — Competency questions for bridging the KnowWhereGraph and Enslaved schemas.
* `2bridge_kwg_gmo.txt` — Competency questions for bridging the KnowWhereGraph and Geolink Modular Ontology schemas.
* `3bridge_enslaved_currkg_core_scholar.txt` — Competency questions for bridging the Enslaved, Curriculum KG, and rich Core Scholar schemas.
* `3bridge_gmo_currkg_core_scholar.txt` — Competency questions for bridging the Geolink Modular Ontology, Curriculum KG, and rich Core Scholar schemas.
* `3bridge_kwg_currkg_core_scholar.txt` — Competency questions for bridging the KnowWhereGraph, Curriculum KG, and rich Core Scholar schemas.
* `3bridge_kwg_enslaved_core_scholar.txt` — Competency questions for bridging the KnowWhereGraph, Enslaved, and rich Core Scholar schemas.
* `3bridge_kwg_gmo_core_scholar.txt` — Competency questions for bridging the KnowWhereGraph, Geolink Modular Ontology, and rich Core Scholar schemas.
* `consolidated_kgs.txt` — Consolidated competency questions across unrelated schemas.
* `core_scholar_rich.txt` — Competency questions for the rich Core Scholar schema.
* `currkg.txt` — Competency questions for the Curriculum KG schema.
* `enslaved.txt` — Competency questions for the Enslaved schema.
* `gmo.txt` — Competency questions for the Geolink Modular Ontology schema.
* `kwg.txt` — Competency questions for the KnowWhereGraph schema.
* `README.md` — Readme file for this directory.
