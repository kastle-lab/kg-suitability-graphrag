# Bridged KG Selection Prompt Assets

This directory contains JSON assets used to fill KG selection prompt templates with examples and reasoning scaffolds for bridging.

## Directory and File Overview

* `2bridge_currkg_core_scholar.json` — Prompt asset examples for bridging the Curriculum KG and rich Core Scholar schemas.
* `2bridge_currkg_kwg.json` — Prompt asset examples for bridging the Curriculum KG and KnowWhereGraph schemas.
* `2bridge_enslaved_core_scholar.json` — Prompt asset examples for bridging the Enslaved and rich Core Scholar schemas.
* `2bridge_enslaved_currkg.json` — Prompt asset examples for bridging the Enslaved and Curriculum KG schemas.
* `2bridge_gmo_core_scholar.json` — Prompt asset examples for bridging the Geolink Modular Ontology and rich Core Scholar schemas.
* `2bridge_gmo_currkg.json` — Prompt asset examples for bridging the Geolink Modular Ontology and Curriculum KG schemas.
* `2bridge_kwg_enslaved.json` — Prompt asset examples for bridging the KnowWhereGraph and Enslaved schemas.
* `2bridge_kwg_gmo.json` — Prompt asset examples for bridging the KnowWhereGraph and Geolink Modular Ontology schemas.
* `3bridge_enslaved_currkg_core_scholar.json` — Prompt asset examples for bridging the Enslaved, Curriculum KG, and rich Core Scholar schemas.
* `3bridge_gmo_currkg_core_scholar.json` — Prompt asset examples for bridging the Geolink Modular Ontology, Curriculum KG, and rich Core Scholar schemas.
* `3bridge_kwg_currkg_core_scholar.json` — Prompt asset examples for bridging the KnowWhereGraph, Curriculum KG, and rich Core Scholar schemas.
* `3bridge_kwg_enslaved_core_scholar.json` — Prompt asset examples for bridging the KnowWhereGraph, Enslaved, and rich Core Scholar schemas.
* `3bridge_kwg_gmo_core_scholar.json` — Prompt asset examples for bridging the KnowWhereGraph, Geolink Modular Ontology, and rich Core Scholar schemas.
* `default.json` — Generic fallback SPARQL prompt asset.
* `README.md` — Readme file for this directory.
