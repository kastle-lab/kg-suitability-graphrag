# KG Selection Prompt Assets

This directory contains JSON assets used to fill KG selection prompt templates with examples and reasoning scaffolds.

## Directory and File Overview

* `core_scholar_rich.json` — Prompt asset examples for the rich Core Scholar schema.
* `currkg.json` — Prompt asset examples for the Curriculum KG schema.
* `default.json` — Generic fallback SPARQL prompt asset.
* `enslaved.json` - Prompt asset examples for the Enslaved schema.
* `gmo.json` - Prompt asset examples for the Geolink Modular Ontology schema.
* `kwg.json` — Prompt asset examples for the KnowWhereGraph schema.
* `README.md` — Readme file for this directory.
