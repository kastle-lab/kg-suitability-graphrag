# Prompts

This directory contains Python prompt template modules used to assemble LLM requests for KG selection.

## Directory and File Overview

* `kg_mul_sel_0_shot.py` — Zero-shot system and user prompt templates for bridging multiple schemas.
* `kg_mul_sel_cot.py` — Chain-of-thought-style prompt template that asks the model to reason internally before returning one query for bridging multiple schemas.
* `kg_sel_0_shot.py` — Zero-shot system and user prompt templates.
* `kg_sel_cot.py` — Chain-of-thought-style prompt template that asks the model to reason internally before returning one query.
* `README.md` — Readme file for this directory.
