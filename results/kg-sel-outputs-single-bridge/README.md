# Single Bridge Results

This directory contains KG selection and evaluation outputs for a single schema.

## Directory and File Overview

* `kg_sel_single_bridge_row_results.csv` — Per-query evaluation results in CSV format.
* `kg_sel_single_bridge_summary.csv` — Aggregated evaluation summary in CSV format.
* `kgsel_0shot_temp0_nen_axiom_ttl_falsesets.jsonl` — Raw JSONL batch prediction output from a model run with zero-shot prompting and competency questions unrelated to the schemas.
* `kgsel_0shot_temp0_nen_axiom_ttl` — Raw JSONL batch prediction output from a model run with zero-shot prompting.
* `kgsel_cot_temp0_nen_axiom_ttl_falsesets.jsonl` — Raw JSONL batch prediction output from a model run with Chain-of-thought-style prompting and competency questions unrelated to the schemas.
* `kgsel_cot_temp0_nen_axiom_ttl.jsonl` — Raw JSONL batch prediction output from a model run with Chain-of-thought-style prompting.
* `README.md` — Readme file for this directory.
