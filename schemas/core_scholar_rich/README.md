# Core Scholar Rich Schema

This directory contains the rich Core Scholar schema and related experiment artifacts.

## Directory and File Overview

* `axiom.txt` — Text description logic axioms converted from the schema LaTeX file.
* `nen.txt` — Natural-language or normalized schema representation used by the experiment pipeline.
* `README.md` — Readme file for this directory.
* `schema.tex` — LaTeX description logic export for the schema.
* `schema.ttl` — Turtle ontology file for the schema.
